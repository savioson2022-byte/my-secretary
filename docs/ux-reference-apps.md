# UX 참고 앱 10개

`나의 비서`는 입력 속도, 반복 사용, 일정 신뢰감이 중요하다. 아래 앱들의 장점을 MVP에 맞게 작게 가져온다.

## 참고 앱과 적용 방향

| 앱 | 참고할 점 | 적용한/적용할 방향 |
| --- | --- | --- |
| Instagram | 로그인 후 내 프로필이 계속 보이는 감각 | 상단 사용자 배지, 원형 프로필 표시 |
| Google Calendar | 일정이 시간 축에서 바로 이해됨 | 주간 캘린더 10분 단위 조정 |
| Apple Calendar | 모바일 기본 캘린더처럼 단순한 입력 | 터치 드래그, 시간 선택 단순화 |
| Todoist | 빠른 기록과 나중 정리 | 홈의 빠른 입력, AI 분류 후 저장 |
| Notion | 기록과 데이터베이스가 같이 보임 | 저장된 기록/일정/장소 분리 |
| Slack | 상태가 명확한 메시지 피드백 | 로그인/이동시간 API 상태 메시지 |
| Uber | 출발지, 도착지, 이동수단 중심 흐름 | 이동시간 캐시 키: 출발지+시각+도착지+수단 |
| KakaoMap | 장소 검색과 길찾기 맥락 | 지도 검색 버튼, 나중 API 자동 계산 |
| iOS Reminders | 알림과 할 일의 간결함 | 알림/PWA 확장 준비 |
| ChatGPT | 자연어 입력으로 작업 시작 | 음성/텍스트 입력 후 AI 분류 |

## 이번 작업에서 반영한 UX 원칙

- 로그인 전에는 회원가입과 로그인을 분리해서 사용자가 현재 흐름을 알 수 있게 한다.
- 로그인 후에는 사용자 이름과 현재 기기 연결 상태를 즉시 보여준다.
- 외부 API 결과는 같은 조건이면 다시 계산하지 않고 캐시한다.
- 모바일에서 입력과 선택은 세로 흐름을 우선하고, 데스크톱에서는 넓은 화면을 활용한다.
- 권한이나 외부 콘솔 설정이 필요한 일은 앱 안에서 실패하지 않고 안내 메시지로 남긴다.
