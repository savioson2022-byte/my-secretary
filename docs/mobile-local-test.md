# 모바일 로컬 테스트

아이폰에서 로컬 개발 중인 `나의 비서`를 확인할 때의 절차다.

## 접속 조건

- Mac과 iPhone이 같은 Wi-Fi에 연결되어 있어야 한다.
- 개발 서버는 `127.0.0.1` 전용이 아니라 같은 네트워크에서 접근 가능한 주소로 실행되어야 한다.
- 현재 Mac의 확인된 Wi-Fi 주소는 `172.30.1.84`다.

## 실행 주소

개발 서버를 네트워크 공개 모드로 실행했다면 iPhone Safari에서 아래 주소를 연다.

```text
http://172.30.1.84:3001/
```

음성 기록을 바로 테스트하려면 아래 주소를 연다.

```text
http://172.30.1.84:3001/?voice=1
```

## iPhone에서 앱처럼 쓰기

1. Safari에서 위 주소를 연다.
2. 공유 버튼을 누른다.
3. 홈 화면에 추가를 선택한다.
4. 홈 화면 아이콘으로 다시 실행한다.

`manifest.webmanifest`에는 `standalone` 표시 모드와 `/?voice=1` 바로가기 정보가 들어 있다.

## 주의

- `http://127.0.0.1:3001/`은 Mac 자기 자신을 뜻하므로 iPhone에서는 열리지 않는다.
- Wi-Fi가 바뀌면 Mac의 주소도 바뀔 수 있다.
- 배포 후에는 Vercel 같은 공개 URL을 사용하면 이 과정 없이 바로 접속할 수 있다.
